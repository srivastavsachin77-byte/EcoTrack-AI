# 🌍 EcoTrack-AI

EcoTrack-AI is a simple web application that helps users estimate their daily carbon footprint and receive eco-friendly suggestions powered by rule-based AI logic.

---

## 🚀 Features
- Calculate footprint based on **transport, electricity, and food** usage.
- Get **AI-powered suggestions** to reduce emissions.
- Secure input validation (no negative or invalid values).
- Accessible design with semantic HTML and labels.
- Unit tests for reliability.
